# 🛡️ КРАТКОЕ РУКОВОДСТВО ПО ВАЛИДАЦИИ

## ✅ ЧТО ИСПРАВЛЕНО

### 🔧 Все баги устранены!

1. **Имя** - теперь нельзя вводить цифры
2. **Возраст** - только числа от 14 до 100
3. **Время** - строгий формат HH:MM
4. **Дата** - проверка корректности
5. **Услуга** - разные тексты для тату и пирсинга
6. **Чек** - обязательно фото

---

## 📋 ПРИМЕРЫ ВАЛИДАЦИИ

### Имя:
```
✅ "Давид" - принято
✅ "Анна-Мария" - принято
❌ "дфава314132453215" - отклонено (цифры)
❌ "123" - отклонено
```

### Возраст:
```
✅ "25" - принято
✅ "18" - принято
❌ "13" - отклонено (младше 14)
❌ "abc" - отклонено (буквы)
```

### Время:
```
✅ "16:00" - принято
✅ "14:30" - принято
❌ "16.03149314" - отклонено
❌ "25:00" - отклонено
```

### Дата:
```
✅ "20.05" - принято
✅ "20 мая" - принято
❌ "32.05" - отклонено (день > 31)
❌ "20.13" - отклонено (месяц > 12)
```

## 📅 НАПОМИНАНИЯ

3. **Через 90 дней** - напоминание о коррекции тату
