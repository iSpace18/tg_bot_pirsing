## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Как реализовано:

1. **Одиночное фото** (555.jpg):
```python
try:
    photo = FSInputFile("555.jpg")
    await safe_send_message(message, text, photo=photo, reply_markup=menu)
except FileNotFoundError:
    await safe_send_message(message, text, reply_markup=menu)
```

2. **Группа фотографий** (123.jpg, 345.jpg и т.д.):
```python
try:
    media = []
    for path in ["123.jpg", "345.jpg"]:
        try:
            media.append(types.InputMediaPhoto(media=FSInputFile(path)))
        except FileNotFoundError:
            continue
    if media:
        await safe_send_media_group(message, media)
except Exception as e:
    logger.error(f"Ошибка при отправке фото: {e}")
```

### Обработка ошибок:
- Если фотография не найдена, бот продолжит работу без неё
- Ошибки логируются в файл `bot.log`
- Пользователь получит текст, даже если фото отсутствует
